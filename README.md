# LIN Frame Detection for ESP32s

This is a project that allows you to connect and listen to a LIN network for frames related to button presses - this is the tool I created and use to detect button presses on a LIN2 Steering Wheel to reverse engineer it into an older vehicle.

It's designed to run on a standard ESP32, however it could be re-purposed to run on most ESP32 boards.

## Project

- Board: ESP32 (Amazon Link [https://www.amazon.co.uk/dp/B0DGLCWR76](https://www.amazon.co.uk/dp/B0DGLCWR76))
- IDE: VS Code
- Extension: PlatformIO
